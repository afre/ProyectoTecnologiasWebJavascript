¿Qué es ECMAScript?

ECMAScript es el nombre del estándar para el lenguaje JavaScript. Su número es ECMA-262. Otras implementaciones de ECMAScript incluyen JScript y ActionScript. Puede también ser entendio como una especificación estandarizada de un lenguaje de scripting.

La sexta edición de ECMAScript fue publicada en junio de 2015. Este añade nuevas sintáxis para la escritura de aplicaciones complejas, pero las define semánticamente del mismo modo que ECMAScript 5. No todos los navegadores soportan ECMAScript 6, sin embargo, es posible transcompilarlo a ECMAScript 5, el cual es soportado por la mayoría de browsers.
